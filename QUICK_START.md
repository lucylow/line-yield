### 🔧 Development Commands

```bash
# Start LIFF app (localhost:3000)
cd packages/liff-app
pnpm run dev

# Start web app (localhost:5173)
cd packages/web-app
pnpm run dev

# Build all packages
pnpm run build:all
```
